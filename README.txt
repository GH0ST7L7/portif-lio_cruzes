PORTFÓLIO • RIBEIRÃO DOS CRUZES

Como trocar rapidamente as fotos
--------------------------------
1) Abra a pasta `assets/images`.
2) Substitua os arquivos mantendo os nomes:
   cruzes1.png (usado no topo/hero e também como logo)
   cruzes2.jpg
   cruzes3.jpg
   ...
   cruzes12.jpg
3) Use o mesmo formato da extensão (PNG ou JPG).

Cores principais (em styles.css)
--------------------------------
--brand:   #0b3b8e  (azul)
--brand-2: #ffd23f  (amarelo)
--bg:      #0a0f1a  (fundo escuro)

Personalização rápida
---------------------
- Título do site: edite a <title> no index.html
- WhatsApp/E-mail: altere os links na seção #contato
- Textos das seções: edite os parágrafos em index.html
- Ícone do site: substitua `assets/icons/favicon.png`

Estrutura
---------
index.html
styles.css
script.js
assets/
  images/  (cruzes1..cruzes12)
  icons/   (favicon.png)

Feito para rodar direto no VSCode: abra a pasta e use Live Server (se quiser recarregar automaticamente).
